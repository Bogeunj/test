# Tetris MVP

브라우저에서 실행 가능한 테트리스 게임입니다.

## 주요 기능

- 10x20 보드
- 7-bag 랜덤 생성 + `seed` 기반 재현 가능한 순서
- SRS 회전 규칙(wall kick 포함)
- 소프트/하드 드롭, 라인 클리어, 점수/레벨/속도 증가
- 홀드(`C`) / 고스트 피스
- 일시정지(`P`), 재시작(`R`), 새 시드 시작(`N`)

## 실행 방법

1. 프로젝트 루트에서 `index.html` 파일을 브라우저로 엽니다.
2. 필요하면 URL에 `?seed=your-seed`를 붙여 같은 블록 순서를 재현할 수 있습니다.

## 조작법

- `←` `→`: 좌우 이동
- `↑` / `X`: 시계 회전 (CW)
- `Z`: 반시계 회전 (CCW)
- `↓`: 소프트 드롭
- `Space`: 하드 드롭
- `C`: 홀드
- `P`: 일시정지
- `R`: 같은 시드로 재시작
- `N`: 새 시드로 시작

## 테스트

- `node --check script.js`
- `node --check tetris-core.js`
- `node --test tests/tetris-core.test.js`

### 단위테스트 커버리지

- 충돌 판정(경계/겹침)
- 라인 클리어 + 점수/레벨/속도 증가
- SRS wall kick 회전
- 홀드 1회 제한
- 시드 재현 + 7-bag 생성
- 스폰 충돌 게임오버
- top-out(보드 상단 잠금) 게임오버
